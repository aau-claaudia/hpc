HPC Cost Estimator

The HPC Cost Estimator is a tool for AAU researchers and employees in fundraising to estimate costs for High-Performance Computing (HPC) projects. It allows users to input project details like duration, storage, compute power, and data management needs. An interactive quiz helps determine optimal compute configurations based on project specifics. This tool aids in planning and budgeting HPC resources efficiently.
