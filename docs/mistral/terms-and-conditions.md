##### Use of Mistral Le Chat and AI Studio at Aalborg University (AAU)

---

## 1. Who may use Mistral at AAU

Mistral Le Chat and AI Studio are provided by Aalborg University (AAU) for **academic research activities only**. The services must not be used for any teaching, commercial or private purposes.


## 2. API keys

API keys issued through AI Studio are **personal and must not be shared** with other individuals. Users are responsible for all activity performed using their API key. 

For security reasons, all API keys must have a maximum lifetime of **30 days**. After 30 days, the key must be replaced with a new one. This policy helps reduce the risk of leaks and unauthorized use of AAU-issued keys. Users are responsible for monitoring key expiration and ensuring timely replacement.

If the user suspects that their API key has been compromised or misused, they must immediately revoke the key and notify CLAAUDIA. CLAAUDIA reserves the right to revoke API keys at any time if misuse, sharing, or security concerns are identified.



## 3. Data classification

!!! warning "No personal data allowed"
    Mistral AI may be used only for data classified as level 1 under [AAU’s data classification model](https://www.sikkerhed.aau.dk/dataklassifikation){__blank}. It is not approved for processing personal data, so data containing personal information must not be uploaded to or processed in Mistral AI. 

    Personal data and data classified as **Level 2 or Level 3 may never** be uploaded, entered, or processed in Mistral Le Chat or AI Studio.

Users are personally responsible for ensuring compliance with AAU's data classification requirements before submitting any content to the service.


## 4. Shared libraries

Users who upload documents to a shared library are solely responsible for ensuring that all library members are authorised to access the uploaded material. Users must not invite individuals to a shared library unless those individuals are permitted to access all documents contained within it.


## 5. Compliance with mistral terms

Use of the services must comply with the applicable [terms and conditions](https://chat.mistral.ai/legal/terms) of Mistral AI in addition to these AAU-specific terms.


## 6. Service availability

Le Chat and AI Studio are provided **"as is"** without guarantees of availability, performance, or uptime.

### Seat Availability

Please note that access to Mistral Le Chat and AI Studio depends on seat availability. CLAAUDIA cannot guarantee access for every researcher at all times.

## 7. Support and responsibilities

CLAAUDIA is responsible for the institutional setup of Mistral at AAU. This includes managing access, workspaces, and token quotas. Specifically, CLAAUDIA can help with:

- Access and login to Le Chat and AI Studio
- AI Studio workspace creation and workspace membership
- Requesting or increasing a token quota

For all technical questions about the Mistral products themselves — including API behaviour, model output, interface issues, and bugs — users must contact **Mistral's support** directly.

## 8. Fair usage 

All AI Studio workspaces have a standard usage qouta per month to ensure a fair usage and make the service availabe for all AAU users. CLAAUDIA can adjust the montly qouta at all times depending on the usage demand form the users. 


## 9. AI Output Disclaimer

**Always verify AI-generated content:** Outputs generated by Mistral Le Chat or AI Studio may be inaccurate, incomplete, or misleading. Users are responsible for verifying any information before relying on it in research, publications, or decision-making.


## 10. Changes to Services

AAU reserves the right to modify, redesign, disable, or remove any part of the services. Where possible, users will be informed of major changes **6 weeks in advance**.

Updates, upgrades, and scheduled maintenance are not considered major changes.



## 11. Changes to Terms

AAU may update these Terms and Conditions periodically. Users will be informed of significant changes.



## 12. Suspension

Access may be suspended or removed if these terms, AAU policies, or Mistral AI's own terms are violated.
