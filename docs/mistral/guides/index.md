# Mistral Guides

This guide section is organized around the two main Mistral products covered at AAU:

* [Le Chat](/mistral/guides/le-chat/) for interactive conversations, drafting, exploration, and quick assistance
* [AI Studio](/mistral/guides/ai-studio/) for structured prompt work, model experimentation, and repeatable setups

## Start here

If you are new to the service, the recommended reading order is:

1. [How to access Mistral](/mistral/how-to-access/)
2. [Le Chat](/mistral/guides/le-chat/)
3. [AI Studio](/mistral/guides/ai-studio/)
