# Le Chat

## What is Le Chat?

Le Chat is Mistral's conversational interface. It is the best place to start when you want quick AI assistance in a browser-based chat format.

For most users, Le Chat will be the easiest entry point for:

* drafting text
* rewriting or summarizing content
* brainstorming ideas
* asking exploratory questions
* getting help with simple coding or technical explanations

## When to use Le Chat

Le Chat is a good fit when your task is interactive and lightweight. Typical examples include:

* getting a first draft of an email, abstract, or meeting summary
* reformulating text for a different audience or tone
* turning rough notes into a cleaner structure
* asking follow-up questions to refine an answer step by step
* exploring a new concept before reading deeper source material

## A simple workflow

1. Start with a short description of your goal.
2. Add the context Mistral needs, such as audience, format, tone, or constraints.
3. Ask for a concrete output format such as bullet points, a table, or a short summary.
4. Refine the result by asking follow-up questions.
5. Review the answer critically before using it in your work.

## Good practices

* Be specific about what you want.
* State the intended audience and desired output format.
* Break large tasks into smaller requests.
* Ask the model to explain assumptions when accuracy matters.
* Verify factual claims, references, and technical details independently.

## Important limitations

!!! warning "Do not paste sensitive data"
    Unless the AAU Mistral setup is explicitly approved for sensitive data, treat Le Chat as unsuitable for confidential, personal, or otherwise restricted information.

!!! info "Use AI Studio for more structured experimentation"
    If you need to compare prompts, work more systematically with instructions, or build a reusable setup, [AI Studio](/mistral/guides/ai-studio/) is likely the better fit.
