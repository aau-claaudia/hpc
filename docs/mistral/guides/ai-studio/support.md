# Mistral Support

This page explains where to get help depending on your question.

---

## What CLAAUDIA can help with

CLAAUDIA's support for Mistral is limited to institutional and access-related topics:

- **Le Chat** — access, seat assignment and login. 
- **AI Studio** workspace creation, workspace members and allocation of budget.

### How to contact CLAAUDIA

| Channel | Details |
|---|---|
| **Service portal** | [serviceportal.aau.dk](https://serviceportal.aau.dk/serviceportal?id=emp_taxonomy_topic&topic_id=82a253e8838fc21053711d447daad328) |
| **Web** | [researcher.aau.dk/contact/claaudia](https://www.researcher.aau.dk/contact/claaudia) |

## Mistral Support (Technical Questions)

For all technical questions — API errors, model behaviour, interface bugs, feature availability, or anything product-related — use Mistral's own support resources.

### Documentation

Mistral maintains a single documentation site covering both Le Chat and AI Studio:

| Resource | URL | Description |
|---|---|---|
| Documentation hub | [docs.mistral.ai](https://docs.mistral.ai/) | Models, capabilities, Le Chat, AI Studio, agents, and deployment |
| API reference | [docs.mistral.ai/api](https://docs.mistral.ai/api) | Full API reference and endpoint specs |
| Cookbooks | [docs.mistral.ai/cookbooks](https://docs.mistral.ai/cookbooks/) | Practical code examples for common use cases |

## Common Issues

| Problem | Where to go |
|---|---|
| Cannot log in to Le Chat | Contact CLAAUDIA via the service portal |
| Cannot access AI Studio workspace | Contact CLAAUDIA via the service portal |
| Token quota exceeded | Contact CLAAUDIA to request an increase |
| API key not working | Check you are in the correct workspace. See [Mistral API docs](https://docs.mistral.ai/) |
| API error (401 unauthorized) | Verify your API key is valid and from the correct workspace |
| API error (429 rate limit) | Add retry logic. See [Mistral rate limits documentation](https://docs.mistral.ai/) |
| Model giving unexpected results | See [Workflows and Tips](/mistral/guides/le-chat/workflows-and-tips/) and [Mistral Discord](https://discord.gg/mistralai) |
| Questions about Le Chat features | See [Le Chat documentation](https://docs.mistral.ai/le-chat/overview/) |
| Questions about AI Studio features | See [AI Studio / API documentation](https://docs.mistral.ai/) |
| Bug report or feature request | Contact Mistral at [mistral.ai/contact](https://mistral.ai/contact) or [Discord](https://discord.gg/mistralai) |
| Data classification questions | See [Terms and Conditions](/mistral/terms-and-conditions/) |
