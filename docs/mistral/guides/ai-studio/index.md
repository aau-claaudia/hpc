# AI Studio

AI Studio is Mistral's developer platform, available at [console.mistral.ai](https://console.mistral.ai){target="_blank"}. It is designed for structured experimentation, API integration, and building repeatable AI-powered workflows. It includes a Playground for testing models interactively, API key management, usage monitoring, and tools for building and deploying agents.

At AAU, basic access to AI Studio (Playground, Document AI, Audio transcription) is available through the same SSO login as Le Chat. API key access requires an additional approval step — see [How to Access Mistral](/mistral/how-to-access/) for details.

For a full overview of AI Studio features, see Mistral's official documentation:

- [AI Studio — Documentation](https://docs.mistral.ai/deployment/ai-studio){target="_blank"} — Guides and reference for the console, API, and deployment options
- [AI Studio — Help Centre](https://help.mistral.ai/en/collections/789620-mistral-ai-studio){target="_blank"} — FAQs for accounts, workspaces, and common issues

## Access at AAU

AI Studio is available to AAU researchers through SSO login using your AAU credentials.

**Basic access** (Playground, Document AI, Audio) is available immediately after login — no application needed.

**API key access** requires a workspace to be set up by CLAAUDIA. Contact CLAAUDIA via the [service portal](https://serviceportal.aau.dk/serviceportal?id=emp_taxonomy_topic&topic_id=82a253e8838fc21053711d447daad328){target="_blank"} to request a workspace.

!!! info "Seat availability"
    Access depends on seat availability. CLAAUDIA cannot guarantee access for every researcher at all times.

---

## Workspace quota

Each workspace has a monthly usage quota of **€50**. This covers API usage (tokens consumed across all models) within that workspace for the month. Note that model prices vary significantly — larger and more capable models consume the quota faster than smaller ones. See [Mistral's pricing page](https://mistral.ai/pricing){target="_blank"} for a full overview.

---

## Data Classification

!!! warning "Level 1 data only"
    AI Studio must only be used with data classified as **Level 1** under [AAU's data classification policy](https://www.security.aau.dk/data-classification){target="_blank"}. Do not enter personal data, confidential research findings, or any sensitive information into prompts or uploaded documents. See [Terms and Conditions](/mistral/terms-and-conditions/) for full details.
