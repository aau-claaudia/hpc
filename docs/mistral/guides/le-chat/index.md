# Le Chat

Le Chat is Mistral's conversational AI assistant, available at [chat.mistral.ai](https://chat.mistral.ai){target="_blank"}. It is designed for interactive, browser-based use — you write a message, the model responds, and the conversation continues.

Le Chat is suited for everyday AI tasks such as drafting, summarising, explaining, translating, brainstorming, and exploring ideas. It includes features like web search, document libraries, image generation, code interpreter, agents, and canvas editing.

For a full overview of features and how to use them, see Mistral's official documentation:

- [Le Chat — Help Centre](https://help.mistral.ai/en/articles/347531-an-overview-of-le-chat-s-capabilities){target="_blank"} — FAQs and guides for all Le Chat features.


!!! warning "Level 1 data only"
    Le Chat must only be used with data classified as **Level 1** under [AAU's data classification policy](https://www.security.aau.dk/data-classification){target="_blank"}. Do not enter personal data, confidential research findings, or any sensitive information. See [Terms and Conditions](/mistral/terms-and-conditions/){target="_blank"} for full details.


## Libraries  

In Le Chat you can create [libraries](https://help.mistral.ai/en/articles/347582-what-are-libraries-and-how-do-i-use-them-in-le-chat){target="_blank"} and upload files to interact and collaborate on them with other AAU Mistral users.

!!! warning "You are responsible for what you share"
    The user who uploads a document to a library is solely responsible for ensuring that all library members are authorised to access it. Only invite people who are permitted to see the documents. We recommend adding users as **Viewers** unless they specifically need upload rights.

Libraries have the following limits:

- **Maximum file size:** You can upload files up to **100 MB** per file.
- **Files cannot be downloaded** from the library back to your computer. Always keep the original file on your own machine.
