# Getting started with Le Chat

## A simple workflow

1. Start with a short description of your goal.
2. Add the context Mistral needs, such as audience, format, tone, or constraints.
3. Ask for a concrete output format such as bullet points, a table, or a short summary.
4. Refine the result by asking follow-up questions.
5. Review the answer critically before using it in your work.

## When to use Le Chat

Le Chat is a good fit when your task is interactive and lightweight. Typical examples include:

* getting a first draft of an email, abstract, or meeting summary
* reformulating text for a different audience or tone
* turning rough notes into a cleaner structure
* asking follow-up questions to refine an answer step by step
* exploring a new concept before reading deeper source material
