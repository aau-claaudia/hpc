⚠️ Important Notice: Strato and UCloud Virtual Machines ⚠️

Due to technical difficulties on our OpenStack platform, users may currently experience periodic issues when attempting to start or stop virtual machines on Strato or the UCloud platform. We are investigating the issues, and will update the status as soon as possible. If a server start up period takes more than 3 minutes, we request users to cancel the job on UCloud, or to delete the instance (server) on Strato.

This problem is not consistent, and it may be possible to successfully build a virtual machine on both platforms if you try to build the same instance multiple times.
