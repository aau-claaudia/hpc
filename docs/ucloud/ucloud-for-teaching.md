# UCloud for teaching at AAU

UCloud is now becoming available for teaching activities at Aalborg University. This gives teachers and students more opportunities to include computing power, AI tools and digital methods in courses, exercises, and project work.

UCloud can help you solve different types of tasks, such as analysing large datasets, using programming environments, testing AI tools, working with language models, recording and transcribing audio, or collaborating with others on shared files. At the same time, students gain hands-on experience with digital workflows and tools that can be useful beyond their studies.

UCloud is a user-friendly platform where you can work with data directly in the browser through pre-installed apps and software. The platform has a graphical interface, which can make it easier to get started compared to more technical HPC platforms. UCloud can also be used for working with data in a secure environment, including [sensitive data](/ucloud/guides/sensitive-data-on-ucloud/). This makes UCloud relevant for teaching activities across disciplines and technical levels.

!!! tip "New to UCloud?"
    CLAAUDIA runs **introductory webinars** for teachers and students (see [Get started with UCloud](#get-started-with-ucloud)). You can also follow the [guides and support](#guides-and-support) section below at your own pace.

## Spend less time on technical issues and more time on learning

With UCloud, you get access to a standardised setup directly in the browser, making it easier to get started. Instead of relying on different computers, operating systems, and software versions from student to student, everyone can work in the same environment, leaving more time to focus on learning and less time spent on technical issues.

## Examples of use cases

UCloud can support teaching activities where students need access to computing power, digital tools or a shared and secure environment for working with data.

Examples include:

- [x] analysing larger datasets than a regular laptop can handle
- [x] working with programming and data analysis, for example in Python or R
- [x] testing AI tools and language models
- [x] creating a chatbot based on course material
- [x] recording and transcribing audio
- [x] visualising data
- [x] labelling visual data
- [x] training AI models
- [x] collaborating on shared files, data and workflows
- [x] working with data in a secure environment

## Apps and software directly in the browser

One of the advantages of UCloud is that you can access apps and software directly in the browser. This means that many tools can be used without installing software locally.

UCloud also gives access to AI-related tools in an HPC environment, making it possible to explore AI workflows alongside other tools for data analysis, programming and digital methods.

UCloud includes apps across different disciplines and use cases. Examples include:

| App | Purpose |
| --- | --- |
| [Transcriber](/ucloud/guides/application-guides/transcriber/) | Transcription of audio |
| [Dictaphone](/ucloud/guides/application-guides/Dictaphone/) | Recording and transcribing audio |
| [ChatUI](/ucloud/guides/application-guides/ChatUI/) | Working with large language models |
| JupyterLab | Notebooks, programming, and data analysis |
| RStudio | R-based data analysis |
| MATLAB | Numerical computing, modelling, and analysis |
| CVAT | Annotation of visual data |
