# Providers on UCloud
In the new UCloud setup there is only 1 provider, the SDU/k8s, to ensure improved user experience by providing the same functunality for all users. 


## SDU/K8s

SDU/K8s is a Kubernetes-based environment designed for scalable research workloads where users can quickly launch pre-built applications.

- **SDU/K8s** provides:
  - ✅ Handling of **sensitive data**  
  - ✅ **SSH connections**  
  - ✅ Support for **licensed software**  
  - ✅ **Fixed IP addresses**
  - ✅ **Virtual machines**
